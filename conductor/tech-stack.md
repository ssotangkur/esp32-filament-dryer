# Tech Stack

## Programming Languages:
*   C/C++ (for the ESP-IDF embedded firmware)
*   Python (for scripting and build tools)
*   JavaScript (for the web frontend and OTA server)

## Frameworks & Libraries:
*   **Embedded:** ESP-IDF, FreeRTOS, LVGL (for UI), LittleFS (for file system)
*   **Web Frontend:** React, Vite
*   **Web Backend (OTA Server):** Node.js, Express
*   **Build System:** CMake
*   **Unit Testing:** Unity, CMock

## Hardware:
*   LilyGo T-Display S3 (ESP32-S3 microcontroller)
